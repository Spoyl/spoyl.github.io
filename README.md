# berkana-landing

The official landing and email capture page for **berkana**, a manufacturer of high quality, portable acoustic drums.

## Project Overview

This repository hosts a minimalist, performance-oriented static email sign up page designed. The focus is on rapid load times, and professional aesthetics.

- **Stack:** HTML5, CSS3 (Vanilla)
- **Analytics:** Plausible (Privacy-focused)
- **Mailing List:** MailerLite
- **Hosting:** GitHub Pages

## Deployment

The site is automatically deployed via the `main` branch.

To make changes locally:

1. Clone the repository.
2. Edit `index.html` or `style.css`.
3. Preview by opening `index.html` in any modern browser.

## Assets

- `logo.png`: The proprietary Berkana logo.
- `me.jpg`: Founder profile image (Class 15/16 branding).
